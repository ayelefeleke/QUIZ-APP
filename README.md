# Quiz App

This is a simple and interactive Quiz Application built using HTML, CSS, and JavaScript.

## Features

* Multiple-choice questions
* Instant answer feedback
* Score tracking
* Personalized result messages
* Perfect score congratulations message
* Restart quiz functionality
* Responsive and clean user interface

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## Project Structure

Quiz-App/

├── index.html

├── style.css

├── script.js

├── screenshot.png

└── README.md

## How to Run

1. Download or clone the repository.
2. Open `index.html` in your browser.
3. Start answering the quiz questions.

## Learning Outcomes

This project demonstrates:

* DOM Manipulation
* Event Handling
* Arrays and Objects
* Functions
* Conditional Statements
* Dynamic Element Creation
* CSS Styling
* Responsive Design

## Future Improvements
